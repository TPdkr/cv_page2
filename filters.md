# Filters used for different pictures

As stated in the README file the [tooooools.app](https://www.tooooools.app/effects/dots) is used for images filter. Here I specified the other important info with regard to this resource, namely the exact filter setting used in some sections. This is helpful for the sake of consistency.

## Projects list
This is how I styles the images for the project list page. The most crucial ones are highlighted with bold.
#### pre-processing
- **blur: 5**
- **grain: 0.5**
- gamma: 1
- black point: 0
- white point: 255
#### post-processing
- **canvs size: 500**
- threshold 128
- grid: regular
- grid angle: 0
- min dot size: 1
- max dot size: 10
- corner radius: 4
- step size: 8
- noise: 2
